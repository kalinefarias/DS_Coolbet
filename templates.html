Long Template

Campaigns with extensive contents as poker tournaments and casino races need a long page. This template consist in a long page with block separation. Can include all components. Below need to have Terms and Conditions dropdown.


Goal: Content.


See the Pen Long Template by Kaline (@coolbet) on CodePen.

Short Template

Campaigns with few content as sportsbook and some casino campaigns need a short page. This template consist in one block with two columns. Can include all components, but is usual have first column with information and the second with "How to Win". Below need to have Terms and Conditions dropdown.


Goal: Content.

Welcome Bonus

All countries has a welcome page, the main objective is new customers conversion. This page need to be tested and monitored always. This template consist on a one block only, full screen, and contain title, subtitle and cards. The number off cards can variated. The card contain a stylized prize component, content and cta.

Week Promotion Calendar

This template should be used when the campaign is weekly and has one offer per day. This template can be used two ways. the first one is having a normal top, and a block with seven cards. Each card will contain a image, a label, topic (H3) and content , and a cta. The second form is a normal top, a highlight block showing the offer of the day, and a block with seven cards. Each card will contain a image, a label, topic (H3) and content. The labels will show the weekday.

Notes: You may fill all days Monday to Sunday. The highlight card will set up itself. If the offer change every day more than a week, you need to use the Long Template with the Highlight Offer feature.


Goal: Content.


HUB

Spootbook has your own special page. It is used in the main sport championship like Euros, Copa, Olympics, etc. This landing page contain a special title and subtitle, special offers cards and a dynamic table.


Goal: Content.


Advent calendar

Important dates requires a special offer. The Advent Calendar is a template for campaigns which needs a special offers every day. Most use is for Christmas and Coolbet's birthday. This page contain a beautiful illustration of our bear and a block with band format which show a special cards for the offers. The offers cards should have a background image (illustration) and may open a modal with offer.


Goal: Content.
